API Hari Libur

Tanggal merah di kalender bisa menyenangkan semua orang.

Cara Mengakses API

  • /api untuk mendapatkan daftar hari libur tahun sekarang.
  • /api?month=8&year=2020 untuk mendapatkan daftar hari libur di bulan 8 tahun 2020.
  • /api?month=8 untuk mendapatkan daftar hari libur di bulan 8 tahun sekarang.

Jika mengakses API di luar cara tersebut tetap mengembalikan code status 200 namun isiannya berupa array kosong [].

Bagaimana cara kerja API ini?

API hari libur mengambil data dari kalenderbali.com dan data per tahun ditarik setiap tanggal 1 Desember menggunakan GitHub Actions.
